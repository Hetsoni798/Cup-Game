# 🥤 Cup Order Match Challenge

A simple and fun **JavaScript browser game** where players try to match the hidden order of cups.
The cups shuffle randomly, and your goal is to remember and match the correct sequence.

## 🚀 Features

* Random cup shuffling
* Score tracking (correct guesses & exchanges)
* Reveal target order option
* Clean UI using HTML, CSS, and JavaScript

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* jQuery

## 🎮 How to Play

1. Click **Start Game** to shuffle the cups.
2. Watch the exchanges carefully.
3. Try to match the hidden order of the cups.
4. Use Reveal Target if you want to check the correct order

## 📂 Project Structure

```
index.html   -> Game layout  
style.css    -> Styling  
game.js      -> Game logic  
img/         -> Cup images
```

## 💡 Purpose

This project was created to practice **DOM manipulation, JavaScript logic, and basic game mechanics**.
